# Heart Disease Classification Dashboard

This project is an interactive machine learning web application that predicts the likelihood of heart disease based on patient medical 
attributes.

The application is built using Python and deployed using Streamlit.

## Live Application

https://heart-disease-classification-3ey2cl9osmwucqtjgf6mny.streamlit.app

## Features

- Predict heart disease risk from patient health inputs
- Compare multiple machine learning models
- Evaluate models using test data
- Visualize model performance

The dashboard includes:

- Model accuracy metrics
- Confusion matrix visualization
- Model comparison chart
- ROC curve comparison
- Interactive patient prediction interface
- Dataset preview

## Machine Learning Models

The following models were implemented and compared:

- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- Neural Network (MLP)

Model performance is evaluated using a **train/test split** to ensure predictions are tested on unseen data.

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Streamlit

## Dataset

Cleveland Heart Disease Dataset

Features include:

- Age
- Sex
- Chest Pain Type
- Resting Blood Pressure
- Cholesterol
- Maximum Heart Rate
- Exercise Induced Angina
- ST Depression
- Number of Major Vessels
- Thalassemia

Target variable:
0 = No Heart Disease
1 = Heart Disease
